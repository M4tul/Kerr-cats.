# Kerr-cats.
The repository for datasets and data-generating codes for the article:  https://doi.org/10.48550/arXiv.2606.22641.

Files: 
present_kerr_cat.ipynb -- reproduces figures 2-5 (uses results.zip as a source folder) 

normalization.ipynb -- computes gaussian thresholds of global infidelity, and catability, stores data into results

comparison.ipynb -- computes fidelity and expectation value, Eqs. (24), (25), and (31), stores data into results

catability_vs_infidelity.ipynb -- computes Eqs. (32), and (34), stores data into results, uses data computed by 
normalization.ipynb 

transposition.ipynb -- generates data for Fig. 6 and plot Fig. 6

transposition2.ipynb -- generates data for Fig. 7 and plot Fig. 7

Folders:
results.zip -- contains all the generated data
